Questões sobre responsividade e Mobile-first

O que são media queries?

  1. Funções JavaScript que alteram o layout do site.
  2. Regras CSS que ajustam o estilo com base no tamanho da tela.
  3. Imagens que se adaptam automaticamente ao tamanho da tela.
  4. Tipos de fontes que mudam de acordo com o dispositivo.

Resposta correta: b

Justificativa: As media queries são usadas em CSS para aplicar diferentes estilos dependendo das características do dispositivo, como a largura da tela. Isso permite que o design de um site seja responsivo e se adapte a diferentes tamanhos de dispositivos, como smartphones, tablets e desktops.

Qual media query usar para aplicar estilos CSS somente ao imprimir uma página web?

  1. @media screen
  2. @media print
  3. @media olny-print
  4. @media all

Resposta correta: c

Justificativa: A media query "@media print" é usada para aplicar estilos específicos quando o conteúdo de uma página é impresso.

Como escrever uma media query para aplicar estilos apenas a dispositivos com uma largura de tela de no máximo 768 pixels?

  1. @media screen and (min-width: 768px)
  2. @media screen and (max-width: 768px)
  3. @media screen and (width: 768px)
  4. @media screen and (viewport-width: 768px)

Resposta correta: b

Justificativa: A media query "@media screen and (max-width: 768px)" é usada para aplicar estilos específicos a dispositivos com uma largura de tela de no máximo 768 pixels.

Como aplicar estilos diferentes para telas com largura entre 480 e 1024 pixels?

  1. @media screen and (min-width: 480px) and (max-width: 1024px)
  2. @media screen and (width: 480px) and (width: 1024px)
  3. @media screen and (min-width: 1024px) and (max-width: 480px)
  4. @media screen and (viewport-width: 480px, 1024px)

Resposta correta: a

Justificativa: A sintaxe "@media screen and (min-width: 480px) and (max-width: 1024px)" é usada para aplicar estilos diferentes a telas com largura entre 480 e 1024 pixels.

Como aplicar um estilo específico somente quando a orientação da tela for retrato (portrait)?

  1. @media (orientation: landscape)
  2. @media (orientation: portrait)
  3. @media (min-orientation: portrait)
  4. @media (portrait)

Resposta correta: b

Justificativa: A media query "@media (orientation: portrait)" é usada para aplicar um estilo específico somente quando a orientação da tela for retrato (portrait).
